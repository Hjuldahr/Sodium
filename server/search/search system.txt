Search terms, resolved left to right 
Use ',' for chaining
Ex: 'id:100,200,300'
Use '~' prefix for optional
Ex: '~cat ~dog' --either cat or dog
Use '-' prefix for exclusion
Ex: '-user:bob' --not the user bob
Use '*' for leading to trailing wildcard
Ex: 'red_*' or '*_wolf'
Use '...' for ranges
Ex: 'score:45...80'
Use '+' suffix for above
Ex: 'score:45+' --score of 45 or above
Use '-' suffix for below
Ex: 'score:33-' --score of 33 or below

default is tag searching

user: --posts created by user
fav: --posts favourited by user
comment: --posts commented on by user
vote: --posts voted on by user
up: --posts up voted on by user
down: --posts down voted on by user

id: --posts with id
score: --posts with score count
n_fav: --posts with favourite count
n_comment: --posts with comment count
n_tag: --posts with tag count
n_char: --posts with character count
n_theme: --posts with theme count
n_view: --posts with view count

rating:s rating:safe --posts depicting SFW content
rating:m rating:mature --posts that depict artistic nudity and suggestive activities (borderline NSFW)
rating:a rating:adult --posts that depict explicit sexual intercourse or sensitive subjects (such as PTSD) 

format: --posts with a specific image format

height: --posts with an image with a specific pixel height
width: --posts with an image with a specific pixel width
ratio: --posts with an image with a specific height / width ratio approximated within 2 decimal places.
file_size: --posts with an image with a specific size measured in kilobytes

created: --posts posted at a particular creation time (date posted)
visited: --posts posted at a particular access time (date last clicked on)
updated: --posts posted at a particular modification time (date laste updated)

title: --posts with a particular title
desc: --posts with a particular description
theme: --posts with a particular theme
